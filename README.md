# duke by Lawnce

**Planner to store your needs!**
**Commands possible**

* help
* todo
* deadline
* event
* list
* done
* delete
* find

## Example Usage with examples
Help
`help`

Todo
`todo study CS2103` 

Deadline
`deadline CS2107 assignment /by 21/07/2019 2359`

Event
`event National Handball League /at 30/09/2020 0900`

List
`list`

Done
`done INDEX`

Delete
`delete INDEX`

Find
`find TEXT`
