# User Guide
Ever wanted a dog? Doggo is a virtual dog that does things for you at your command!
You dont even need to feed it or provide a home for it. He provides unconditionally.
 
## Features 
1. Help
    1. `meow`
1. Add task
   1. `todo`
   1. `deadline`
   1. `event`
1. Show task
   1. `list`
   1. `find`
1. Update task
    1. `done`
    1. `delete`
1. Quit
    1. `bye`
